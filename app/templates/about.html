About Mini Doctor AI

An open-source AI symptom checker built with modern machine learning.

🧠

DistilBERT Model

Powered by a fine-tuned DistilBERT transformer model trained on 4,920 disease-symptom examples across 42 disease classes.

📊

High Accuracy

Achieved 99.9% validation accuracy after fine-tuning. Uses softmax probabilities to rank the top 3 most likely conditions.

🔍

Fuzzy NLP

Natural language symptom extraction using rapidfuzz token matching — handles typos, word order, and synonym variations.

📷

Camera Detection

Optional visual symptom detection using a computer vision model to identify visible skin conditions from your camera.

🏛

Open Source

Built with Python, Flask, and Hugging Face Transformers. The full source code and training pipeline are available on GitHub.
